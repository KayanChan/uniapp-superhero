# uniapp-superhero

[官方文档](https://uniapp.dcloud.io/)

仿[超英预告](http://www.imovietrailer.com/#/)做的一个简单demo，熟悉一下HBuilderX的使用&运行多端&发行包的功能


### 运行
项目添加到`HBuilderX`，然后点击菜单栏`运行`-`运行到浏览器`
